# Roteiro de Conversa — VoDiBus!
**Objetivo:** Prospecção de reconhecimento na UBS

---

1. Costuma usar ônibus para se locomover?

2. Tem dificuldade de descobrir qual ônibus é o ideal para ir onde quer?

3. Você usa celular regularmente?

4. Quais aplicativos conhece? (WhatsApp, TikTok, Instagram?)

5. Quais dificuldades tem para usar aplicativos?

6. Seria interessante um aplicativo que te ajudasse a se locomover usando ônibus?

7. Quer ver um que estou desenvolvendo com esse objetivo?

---

**Anote sempre:**
- Idade aproximada de quem respondeu
- Se usa ou não celular
- Expressões que usam para descrever o problema
